plugin_graphviz

Synopsis

plugin_graphviz<bool show_implications = false>

Description

The plugin_graphviz plugin constructs a GraphViz graph, written to the files graph-0.dot, graph-1.dot, etc. (where 0, 1, etc. are the thread indices).

To plot the .dot files, use the dot command like this:

dot -Tpdf < graph-0.dot > graph-0.pdf
