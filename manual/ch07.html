Chapter 7. Designated plugins

Designated plugins are plugins with a certain predefined purpose. For the moment, the class solver takes the following types of designated plugins as its arguments:

Random

Random number generator. Any of the standard C++11 random number generators (e.g. std::minstd_rand, std::ranlux24_base, or std::mt19937) will do.

Decide

Decision literal heuristic.

Propagate

Propagation engine.

Analyze

Conflict analysis engine.

Send

Outgoing-clause sharing heuristic.

Receive

Incoming-clause acceptance heuristic.

NextRestart

Restart policy.

Reduce

Clause database reduction heuristic.

Simplify

Simplification passes. More than one type of simplification pass can be used at a time using simplify_list<>.

Plugin

Generic plugins. More than one generic plugin can be used at a time using plugin_list<>.

Example 7.1. Example solver configuration

using namespace satxx11;

typedef solver<ranlux24_base,
    decide_cached_polarity<decide_vsids<95>>,
    propagate_watchlists<>,
    analyze_1uip<minimise_minisat>,
    send_size<4>
    receive_all,
    restart_multiply<restart_luby, restart_fixed<400>>,
    reduce_size<2>
    simplify_failed_literal_probing,
    plugin_list<plugin_stdio>> my_solver>
