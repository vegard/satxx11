Introduction

In this chapter, you will learn how to write a plugin from scratch and use it with the standard solver. The plugin itself will be a decision variable heuristic based on two existing heuristics, but many of the techniques used for decision variable heuristic plugins will apply to other kinds of plugins as well.

The particular heuristic that we will create is similar to the VSIDS heuristic, except that some predefined percentage of the decisions will be made at random. We can achieve this by reusing the two built-in decision heuristics decide_vsids and decide_random (without the use of copy/paste, of course!).
