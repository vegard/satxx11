Chapter 1. Overview

This is the manual for the software program/project SAT++11 (sometimes written satxx11). SAT++11 aims to be a fast, scalable, and extensible parallel SAT solver. Efficient compile-time extensibility is one of the most important features of the code base, and we achieve it through creative use of C++ template metaprogramming. In fact, we rely on many of the features of the new C++ language standard, C++11, a fact which is reflected in the name of the solver.

The first part of this manual (Part II, “Usage”) describes how to build the solver program and how to run it. The second part (Part III, “Plugin architecture”) is a reference to the plugin architecture that allows one to write compile-time extensions for the solver, including a tutorial that explains the process of writing a plugin step by step. The last part (Part IV, “Built-in plugins”) is a reference to the built-in plugins that are bundled with the solver core.

Although the solver was written from scratch, many algorithms, heuristics, and techniques were heavily inspired by (and sometimes copied from) their respective counterparts in other, already existing SAT solvers like Chaff, MiniSat, CryptoMiniSat, lingeling, and SAT4J. The main contribution of SAT++11 is to bring the extensibility of SAT4J together with the raw efficiency of C++ solvers like MiniSat.
