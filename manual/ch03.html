Chapter 3. Building the documentation

In order to build the documentation, the following prerequisites must be installed:

  • Docbook (docbook, docbook-xsl)

  • Doxygen (doxygen)

  • Libxslt1.1 (libxslt1.1)

  • xmlto (xmlto)

To compile the API reference, run the following command in the top-level project directory:

doxygen

The generated files can be found in doc/html/.

To compile the manual, run the following commands in the top-level project directory:

cd doc && bash make.sh

The generated files can be found in doc/manual/.
