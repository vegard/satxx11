Chapter 2. Building the solver

Before SAT++11 can be compiled, the following prerequisites must be installed:

  • Boost.Program_options (libboost-program-options-dev)

  • Git (git)

  • GNU C++ Compiler (gcc)

To compile the solver, simply run the following command in the top-level project directory:

bash make.sh
